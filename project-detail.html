Portfolio Webseite (M293 Projekt)

Ein modernes, responsives Web-Portfolio mit Filterfunktion, Kontaktformular und ansprechendem Design.

Projektbeschreibung

Diese Portfolio-Webseite wurde im Rahmen des Moduls M293 entwickelt. Ziel war es, einen modernen, responsiven Webauftritt zu gestalten, der Projekte, Skills und Kontaktmöglichkeiten übersichtlich präsentiert. Die Seite bietet eine Projektübersicht mit Filterfunktion, ein Kontaktformular, eine FAQ-Sektion und ein Newsletter-Abo. Das Design ist vollständig responsive und auf eine ansprechende User Experience optimiert.

  • Kategorie: Coding / Webentwicklung
  • Technologien: HTML5, CSS3, JavaScript
  • Features: Filterbare Projektübersicht, Kontaktformular, FAQ, Newsletter, Responsive Design, modernes UI
  • Besonderheiten: Komplett ohne Frameworks, nur mit HTML, CSS und Vanilla JS umgesetzt. Fokus auf Design und Usability.
